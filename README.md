# Anuvind - Personal Portfolio 🚀

A stunning, highly animated personal portfolio website built with pure Vanilla HTML, CSS, JavaScript, and GSAP.

## 🌟 Features
- **GSAP Animations:** Features smooth scroll reveals, dynamic entrance animations, and a floating particle background loop.
- **Glassmorphism UI:** Premium dark-mode styling utilizing deep blacks and translucent frosted-glass project cards.
- **Zero Build Tools:** Extremely lightweight—no `npm` install or build processes required. Just open and edit!

[View Live Deployment Here](https://anuvind-portfolio-demo.surge.sh)

## 🛠️ Running Locally
Since this project uses no build tools, you can view it immediately without setup:
1. Clone this repository to your local machine:
```bash
git clone https://github.com/anuvindkichu-droid/portifolio-website.git
```
2. Simply double-click `index.html` to open it in your browser. 
3. *Optional:* If you want to run it via a local development server for testing, run:
```bash
python -m http.server 8000
```
Then navigate to `http://localhost:8000`.

## 🎨 Customization
- **Information:** Open `index.html` and replace the placeholder text, project descriptions, and `@anuvind_olv` social handles.
- **Themes:** Open `style.css` and tweak the CSS variables at the top (`--accent-color`, `--accent-alt`) to apply a completely new color scheme.
- **Interactivity:** Open `script.js` to change the behavior and timings of the GSAP animation triggers.
